# React Native App - eConceptual

This project is a React Native mobile application that includes the following features:

- Login with Email-password.
- Profile show and edit functionality.
- List rendering with pagination.

## Requirements

- Node.js
- React Native CLI
- Android Studio / Xcode (for running on Android/iOS emulators)

## Installation

1. Clone the repository:

```sh
git clone https://github.com/bkabhi/eConceptual.git
cd eConceptual
```

2. Install the dependencies:

```sh
npm install
```

3. Install the required pods for iOS:

```sh
cd ios
pod install
cd ..
```

## Running the Project

1. Start the Metro bundler:

```sh
npm start
```

2. Open Android emulator:

3. Run the Android project:

```sh
npm run android
```
